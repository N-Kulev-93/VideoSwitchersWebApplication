# About 
This repository must provide solution for a specific problem at home. I have a couple of video matrix switchers from different brands. Some can be controlled only via RS-232, others by Telnet or web API. I want to be able to remotely control these systems using single web application. The problem I want to solve is probably very rare edge case but anyway i'll try to provide flexible solution for any video matrix switcher no matter the brand. The absraction must be able to configure common video matrix switching commands using RS-232, Telnet or API connection and possibly some additional features regarding multi video switch environment... I wish myself good luck!
While solving my specific edge case problem I hope it will bring bonus points for my upcoming job interview which is the main reason im making it publicly available. I clearly understand it not targeting a huge user audience, anyway from engineering standing point it's a solution to an existing problem! I'm using my still in-progress SPA template project as a base.
